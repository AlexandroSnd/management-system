# Система Модерации Объявлений

Веб-приложение, разработанное для симуляции внутренней системы модерации, предназначенной для быстрой и эффективной проверки, управления и отслеживания эффективности работы модераторов с объявлениями пользователей.

# Cтек

- Фронтенд и Логика

  - `React` — Основная библиотека для создания интерфейса.
  - `TypeScript` — Строгая статическая типизация.

- Управление Состоянием и Данными

  - `TanStack Query` — Управление серверным состоянием, кэшированием и синхронизацией.

  - `Axios` — HTTP-клиент для асинхронных запросов и их отмены.

- Стилизация и Компоненты

  - `Tailwind CSS` — Утилитарный фреймворк для быстрого дизайна.
  - `shadcn/ui` — Набор готовых, доступных UI-компонентов.

- Инфраструктура и Роутинг

  - `Vite` — Высокоскоростной сборщик и инструмент для разработки.
  - `react-router-dom` — Декларативная маршрутизация.

- Инструменты Визуализации

  - `Recharts` — Библиотека для создания интерактивных графиков.
  - `React Image Gallery` — Компонент для отображения галереи изображений.

- Качество Кода

  - `ESLint` — Статический анализ для обеспечения чистоты и единообразия кода.

# Основной Функционал

Проект полностью реализует все обязательные и некоторые дополнительные функциональные требования, включая:

## Главная страница — Список объявлений (/list)

- Отображение: Объявления в виде карточек, содержащих Изображение, Название, Цену, Категорию, Дату, Статус и Приоритет.

- Фильтрация и Поиск:

  - Множественный фильтр по Статусу и одиночный по Категории.

  - Фильтр по Диапазону Цен.

  - Поиск по Названию объявления.

  - Кнопка сброса всех фильтров.

  - Сортировка: По Дате создания, Цене и Приоритету.

  - Пагинация: Постраничное отображение по 10 объявлений, навигация и показ общего количества.

  - Фокус по нажатиии кнопки `/`

## Страница детального просмотра (/item/:id)

- Детальная информация:

  - Полное описание, характеристики (таблица), информация о продавце (Имя, Рейтинг, Объявления, Дата регистрации).

- Галерея:

  - Изображения товара с помощью компонента react-image-gallery.

- История Модерации:

  - Полный список действий с датой, модератором и принятым решением.

- Панель действий модератора:

  - Кнопки: «Одобрить» (зеленая), «Отклонить» (красная), «Вернуть на доработку» (желтая).

  - При отклонении — обязательное поле для причины и Быстрые Шаблоны (Запрещенный товар, Неверная категория и др.).

  - Навигация: Кнопки «Предыдущее» / «Следующее» для ускорения процесса модерации c поддержкой управления с клавиатуры (`←`, `→`).

## Страница статистики модератора (/stats)

- Метрики: Блок с ключевыми метриками (Всего проверено, % одобренных/отклоненных, Среднее время проверки).

- Графики (`Recharts`):

  - Столбчатая диаграмма активности по дням.

  - Круговая диаграмма распределения решений.

  - График по категориям проверенных объявлений.

# Запуск

В корне проекта выполните команды:

- Для запуска backend сервера:

```
cd tech-int3-server
npm install
npm start
```

- Для запуска frontend сервера:

```
cd ad-management-system
npm install
npm run dev
```
